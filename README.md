# Numerical methods library

Библиотека на Java 17 для высокоточного нахождения корней нелинейных уравнений. Проект демонстрирует применение численных методов с использованием произвольной точности вычислений.

## Ключевые особенности
* **Высокая точность:** Полная поддержка `BigDecimal` во всей цепочке вычислений.
* **Fluent API:** Гибкая настройка условий задачи через `SolverContext` (кратность корня, лимит итераций).

## Архитектура и паттерны
Проект спроектирован с упором на расширяемость:
1. **Strategy (Стратегия):** Интерфейс `RootSolver` позволяет прозрачно переключаться между алгоритмами.
2. **Fluent Interface:** Организация гибкого контекста вычислений для читаемого кода.
3. **Repository:** Централизованное хранение математических моделей функций `FunctionRepository`.
4. **Static Factory:** Создание контекста задачи (в т. ч. конфигурации решения) через статические методы в `SolverContext`.

## Реализованные методы

| Метод | Описание | Сходимость |
| :--- | :--- | :--- |
| **Bisection** | Метод деления отрезка пополам. Самый надежный, требует смены знака на границах. | Линейная |
| **Newton** | Метод касательных. Эффективен при наличии производной и хорошего приближения, наиболее быстрый. | Квадратичная |

## Документация и запуск
Код сопровождается **Javadoc**. Исходные файлы документации и отчеты API находятся в папке `/docs`.

### Пример использования:
```java
var solver = new NewtonSolver();
var context = SolverContext.forNewton(new BigDecimal("4.0"))
                           .withMultiplicity(2)
                           .withIterations(100);

SolutionResult result = solver.solve(function, context, mathContext);
